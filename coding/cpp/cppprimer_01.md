### 练习1.2

改写程序，让它返回-1。返回值-1通常被当做程序错误的标识。重新编译并运行你的程序，观察你的系统如何处理main返回的错误标识。

**答：**

Mac下编译器clang，返回-1，`./main`没有发现任何异常。

`echo $?`，返回255

### 练习1.3

编写程序，在标准输出上打印Hello, World

**答：**

```c++
#include <iostream>

int main()
{
	std::cout << "Hello, World" << std::endl;
	return 0;
}
```
